# Linux commands

## cd - go to directory

|flags | meaning |
| --- | --- |
| - | go to previous directory |
| .. | go to parent directory |
| ~ | go to home directory |
| -P | use physical structure of directory and ignore synbolic links |
| -L | use synbolic links |

## ls - look what files are located in current directory

| flags | meaning |
| --- | --- |
| -t | type files sorted by time from newest to oldest |
| -1 | type files from new line |
| -l | type files with detail information |
| -h | type files in human-readable form |
| -d | type info about directory |
| -t | type files in order from last modified to first |
| -r | type files in reverse order |
| -a | type all files even hidden |
| -A | type all files even hidden but without '.' and '..' |
| -R | type files recursively (file from subdirectories) |
| -i | type files with inode number |
| -q | type files without control characters (irregular or non-printable) |
